# AGENTS.md

## Обязательные локальные правила для этого репозитория

1. После любых структурных изменений репозитория обязательно обновлять пакет проектной документации.
2. К структурным изменениям относятся:
   - добавление новых файлов или каталогов;
   - удаление файлов или каталогов;
   - переименование;
   - перенос файлов между папками;
   - изменение route-структуры;
   - изменение ownership зон;
   - существенное изменение межмодульных зависимостей.
3. Обязательная команда после таких изменений:
   - `pnpm project:docs:full`
4. Эта команда должна обновить:
   - `PROJECT_TREE.md`
   - `PROJECT_ROUTES.md`
   - `PROJECT_OWNERSHIP.md`
   - `PROJECT_DEPENDENCIES.md`
   - `PROJECT_RISKS.md`
   - audit-артефакты в `audit/`
5. Если генерация не прошла, работа не считается завершенной, пока:
   - либо ошибка не исправлена;
   - либо причина не зафиксирована явно в ответе и в рабочем контексте.
6. При любых следующих сессиях, где меняется дерево проекта или его архитектурная карта, это правило остается обязательным.
